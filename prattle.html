Project Summary
Role - Web Design & Development
Client - Jaden Dessureault (Personal)

With the vision of starting online conversations with friends without relying on any third-party services and intent on learning more about functional web apps, Prattle was born.

Prattle is a minimalist's web app, designed for and easily accessible from any device connected to the internet. It takes advantage of the some of the newer technologies, such as asynchronous polling of JSON data.

The initial goal was to have a message send instantly to all peers as soon as the original author clicks send. However, due both software and resource contraints, this wasn't entirely possible. I had to settle with fetching data via JavaScript polling intervals, which was still entirely new to me. So either way, I was able to experiment and further my understanding.

After learning about technology such as Node.js and how easily push techniques could be implemented with WebSockets, I've started digging. As I continue to learn how Node works, I've got future plans of rewriting this PHP-powered app.

Software & Technology
HTML - Layout structure and data output.
CSS - Responsive design, fonts, styling and subtle animations.
jQuery - User interaction, data polling, notifications and keyboard shortcuts.
PHP - Procedural back-end system, JSON data parsing & serialization.
