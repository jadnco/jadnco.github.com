Project Summary
Role - Branding, Web Design & Development
Client - Jaden Dessureault (School)

XV was a project for my Graphic Arts and Computer Science classes, with the name being my age at the time, in Roman Numerals.

In Graphic Arts, I was challenged to come up with a faux product then produce the logo and several stationary designs such as business cards, envelopes and pens. It was coincidental that I also had to develop some sort of application for my final Computer Science project.

As my fellow classmates were developing simple games in the taught language (Microsoft’s C#), we were also given the opportunity to present a different kind of project and in different languages. That being said, I chose to create something I’ve wanted to create for some time, a simple online blogging platform. This app would allow you to create several accounts and post blog entries with a live markdown editor. This was the first major web project I’d worked on and forced me to dig deeper into some concepts I found very fuzzy such as relational databases, login systems and user-friendly asynchronous I/O.

We weren’t given much time to work on these projects as the semester was almost over, so I didn’t think I’d be able to show a final working product. Fortunately, I did manage to get everything together and I was quite happy with how it turned out.

Software & Technology

InDesign - Final digital case study production.

CSS - Fonts, styling, animations & general visual changes.

MySQL - Database query structure to store user data and posts.
