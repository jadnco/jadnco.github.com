Project Summary
Role - Web Design & Development
Client - Dreamscape Winnipeg

Dreamscape Winnipeg is a unique opportunity for investors; an economic and environmentally friendly amusement park in the heart of the country. I was commissioned to design and develop a website which would attract potential investors and future guests onboard with the initial plans.

The client wanted to showcase large stock images as soon as you reach the website, as it would allow users to easily visualize the potential. I had to implement an easy way to edit the page text, for anyone without technical knowledge. Instead of taking the usual route by using one of the many popular and feature rich content management systems, I decided to make a custom one from scratch. Although, this was a lot more work in the end, it allowed me to build my skill set and a decent library of functions. The CMS is extremely simple to use. The admin hovers over any text and clicks on its edit icon. The content is typed out and once done, you click the green checkmark and the changes would be sent to the database asynchronously, without unnecessary page refreshing.

Software & Technology
HTML - Page structure, video player, and admin interface.
CSS - Responsive design, fonts, styling and subtle animations.
jQuery - Asynchronous data calls, video player controls and animations.
PHP - Custom content management system, data serialization.
MySQL - Store the content and their unique keys.
