Project Summary
Role - Web Design & Development
Client - Jaden Dessureault (Personal)

Showcase was a personal project I've built to collect together several of my projects, or anyone else's, and display them in a minimalistic and user-friendly manner.

Once an account is created, the user has the ability to upload image thumbnails, add titles and link the project to its associated Dribbble and/or Behance page. This time around, I chose to experiment a little by using flagged JSON files to store all the project and user data, as opposed to the usual MySQL database. Once the back-end traverses through the file storing the project data, it's sent as output to the client side.

I've also used this project to experiment with user activity logs, which are extremely cool. The log records when a user logs in, uploads new projects, logs out and if there's been any failed login attempts. Not exactly useful in this case, it was used mainly as a debugging tool, but I can see much greater uses in the fututre, such as custom analytics.

Software & Technology
HTML - Basic site structure, search engine optimization and data output.
CSS - Fonts, styling, animations & general visual changes.
jQuery - User interactions, progress animations and asynchronous data reading.
PHP - Login system, JSON parsing, image uploading and advanced logic.
